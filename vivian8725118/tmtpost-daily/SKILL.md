# 钛媒体每日资讯

抓取钛媒体最新内容，聚合输出当日科技行业要闻，支持智能搜索和深度解读。

## 🆕 首次使用引导

当用户触发 skill 但意图模糊时（如只说"钛媒体"、"能做什么"），先展示以下引导菜单：

```
📰 钛媒体资讯助手

我可以帮你：

1️⃣ 钛媒日报 - 获取今日最新资讯汇总
   说"今日资讯"或"钛媒日报"

2️⃣ 热门榜单 - 查看当前最热的文章排行
   说"热门榜单"或"钛媒体热门"

3️⃣ 关键词搜索 - 搜索你感兴趣的科技话题
   说"搜一下AI手机"或"搜索比亚迪"

4️⃣ 文章解读 - 深度总结任意文章要点
   给我文章链接，然后说"总结一下"

5️⃣ 话题聚合 - 追踪某个话题的多篇报道
   说"聚合一下大模型相关报道"

6️⃣ 分类偏好 - 只看你关心的领域
   说"我关注AI和新能源"

---
直接说你想做什么，或者让我帮你搜索！
```

**注意**：如果用户已有明确意图（如说"钛媒日报"），直接执行，不显示此菜单。

## 核心功能

### 1. 钛媒日报（默认）

获取当日最新资讯汇总。

**工作流程：**

#### 1.1 抓取首页最新文章

使用 web_fetch 工具抓取首页：

```
url: https://www.tmtpost.com/
extractMode: markdown
maxChars: 8000
```

从返回内容中提取前10条文章的标题、URL和发布时间。

#### 1.2 抓取快报资讯

**重要技巧**：快报链接不在 `/nictation` 页面，而是嵌入在**首页（`/`）**的HTML中！

使用 web_fetch 工具**抓取首页**，然后提取快报链接：

```
url: https://www.tmtpost.com/
extractMode: markdown
maxChars: 15000
```

**提取规则**：从首页内容中提取所有 `/nictation/数字.html` 格式的链接，结合快报标题和发布时间。

**为什么不能抓 /nictation**：快报频道页面使用JS动态加载，web_fetch 抓取不到链接。但首页会直接渲染快报链接。

**最终输出格式**：
```
| 时间 | 标题 | 链接 |
| 16:54 | 中证指数修订... | https://www.tmtpost.com/nictation/7941080.html |
```

#### 1.3 抓取热门榜单

使用 web_fetch 工具抓取热门页面：

```
url: https://www.tmtpost.com/hot
extractMode: markdown
maxChars: 8000
```

提取前10条热门文章的标题和URL。

### 2. 关键词搜索

当用户询问特定主题时（如"搜一下AI手机"、"钛媒体有关于新能源的文章吗"），执行搜索。

**工作流程：**

1. 解析用户意图，提取关键词
2. 构搜索URL：`https://www.tmtpost.com/search?q=关键词`
3. 抓取搜索结果页面
4. 提取匹配的文章列表（标题、URL、发布时间）

```
url: https://www.tmtpost.com/search?q=AI手机
extractMode: markdown
maxChars: 8000
```

**输出格式：**
```
🔍 搜索「关键词」结果：

1. [文章标题](URL) - 发布时间
2. ...
```

### 3. 文章深度解读

当用户指定某篇文章或要求总结时，深度读取并总结文章内容。

**工作流程：**

1. 获取文章URL
2. 抓取文章正文

```
url: <文章URL>
extractMode: markdown
maxChars: 20000
```

3. 提取并总结：
   - 文章标题和发布时间
   - 核心观点（3-5个要点）
   - 关键数据/数字
   - 结论摘要

**输出格式：**
```
📄 文章深度解读

**标题**：[标题](URL)
**发布时间**：YYYY-MM-DD

### 核心要点
1. ...
2. ...
3. ...

### 关键数据
- ...

### 总结
...
```

### 4. 话题聚合

当用户关注某个持续性话题时（如"AI手机最新动态"、"比亚迪相关报道"），聚合同话题的多篇文章。

**工作流程：**

1. 搜索话题关键词
2. 筛选近7天的相关文章
3. 按时间排序输出

**触发词**：".topic"、"聚合"、"相关报道"、"追踪"

### 5. 分类早晚报

当用户设置感兴趣的分类偏好时，推送定制化资讯。

**支持的分类：**
- AI/人工智能
- 新能源/电动汽车
- 电商/零售
- 智能手机
- 企业服务/SaaS
- 游戏/元宇宙
- 金融科技
- 半导体/芯片

**设置方式：**
用户说"我关注AI和新能源" → 记录偏好到 USER.md

**推送流程：**
1. 读取用户偏好分类
2. 抓取首页，筛选匹配分类的文章
3. 输出定制化日报

## 错误处理

- 抓取失败时尝试 `/new` 页面
- 内容为空时多抓取几次
- 时间判断以文章发布时间为准
- **快报链接获取**：必须从首页（`/`）提取，不能抓 `/nictation`。快报频道是JS动态渲染，链接不在HTML源码中。
- 搜索无结果时，尝试简化关键词或扩大时间范围
